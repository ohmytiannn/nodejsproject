This is a basic telegram bot that receives an inline query in telegram
and returns a synonym that is related to the input.

It makes use of 3 npm packages: node-telegram-bot-api (https://github.com/yagop/node-telegram-bot-api)
                                bluebird (https://github.com/petkaantonov/bluebird)
                                thesauras (https://github.com/daizoru/node-thesaurus.git")
